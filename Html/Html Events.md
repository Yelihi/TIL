## HTMl Events

<p>사용자가 인풋기기를 통해, 혹은 브라우저가 웹 페이지와 일어난 interaction. 이것을 우리는 이벤트라고 한다. 사용자측에서 일어난 모든 이벤트를 인식할 수 있는 브라우저기능을 사용해 자바스크립트로 그 이후에 순차적으로 일어나게 하고자 하는 일을 발생시킬 때 사용하는 것이다.</p><br />

<p>이벤트의 종류는 너무 많아서 docs 를 참고하는것을 추천한다. 대표적인것들을 몇가지 살펴보자</p>

- onclick : 마우스를 클릭한 후 땔때 작동하는 이벤트
- onmousedown : 마우스를 클릭할 때 작동하는 이벤트
- onblur: focus 가 와서 다음으로 넘어갈 때 작동하는 이벤트
- onchange: input 값이 변화가 있을 때 발동하는 이벤트
- onfocus: focus 가 왔을 때 발동하는 이벤트

<p>이런 방식과는 별개로 자바스크립트로 이벤트를 구독시킬 수 있다. 여기도 정말 무수히 많은 이벤트가 있다. 그럼 html event 와의 차이점은 무엇일까. html event 의 경우 단 하나의 이벤트만 정의가 가능하지만, 자바스크립트를 통하면 여러 이벤트를 구독시킬 수 있다.</p><br />

<p>일반적으로 html 문서 내에 직접 html attribute 를 통해서 이벤트를 작성하는것은, html 에 inline 으로 javascript 를 실행시키는 것이니 피하는것이 좋다. 다만 리엑트와 같이 html 태그를 생성시키는 방식이라면 상관없다</p>
