### 타입스크립트를 학습하는 이유

<p>타입스크립트를 적용시키는 가장 중요한 이유라면, 바로 프로그램의 안정성을 상승시키기 위해 타입스크립트를 사용한다. 타입스크립트는 황당한 에러들을 많이 줄여준다. 다만, 자바스크립트보다 자유도가 줄어들게 된다. 그래도 에러 안나는게 훨씬 좋으니깐 이득이 훨씬 많다.</p>
